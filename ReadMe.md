# BFX Test Assignment

## Step up environment

```sh
npm install
```

## To start the server

```sh
sh server.sh
```

## To start the client

```sh
sh client.sh eth buy 100 5
```

The first arguement is token. eg. eth or bnb
The second arguement is side. eg. buy or sell
The third arguement is price.
The last arguement is quantity.
